# SmartBot - AI 客服 SaaS 项目规划

## 1. 项目概述

**产品名称**: SmartBot  
**一句话定位**: 面向中国出海团队 + 中小企业的 AI 客服 SaaS，按消息计费，5 分钟配置上线

**核心价值**:
- 多平台接入（网站/WhatsApp/微信/小红书）
- RAG 知识库，让 AI 像人工一样精准
- 零代码配置，非技术人员也能用
- 合理定价，按使用量计费

## 2. 市场定位

**目标用户**:
- 中国出海电商/APP 团队（主要）
- 中小企业内勤客服（次要）
- 需要多语言客服的跨境团队

**差异化**:
- 中文后台 + 多语言支持（英语/越南语/印尼语/泰语）
- 小红书私信接入（国内独家）
- 按对话计费，不绑年付
- 比 Intercom/Zendesk 更便宜、更简单

## 3. 功能规划

### MVP (v1.0)
- [x] 网站聊天组件（Web Widget）
- [x] 知识库管理（上传文档，自动分块）
- [x] AI 回复引擎（RAG + LLM）
- [x] 多轮对话 + 意图识别
- [x] 人工接管（AI → 人工无缝切换）
- [x] 基础数据分析（对话量/解决率/满意度）
- [x] 用户注册/登录/订阅

### v1.1
- [ ] WhatsApp Business 接入
- [ ] 微信客服接入
- [ ] 小红书私信接入
- [ ] 邮件通知

### v1.2
- [ ] Slack/Discord 接入
- [ ] 工单系统
- [ ] 团队协作

## 5. 技术架构

### 后端
- FastAPI + SQLAlchemy (PostgreSQL)
- LangGraph (工作流)
- pgvector (RAG 向量搜索)
- Redis (缓存/队列)
- Celery (异步任务)

### 前端
- React 18 + TypeScript
- Tailwind CSS
- ReactFlow (工作流可视化)

### 基础设施
- Vercel/Netlify (前端)
- Railway/Render (后端)
- Supabase (数据库)
- Upstash (Redis)
